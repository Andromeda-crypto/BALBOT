## Overview

BALBOT is a two-wheeled self-balancing robotic platform designed to investigate real-time embedded control, inertial sensing, sensor fusion, and feedback stabilization.

The system combines an ESP32-based control architecture, inertial measurements from the MPU6050, wheel encoder feedback, and closed-loop motor control to stabilize an inherently unstable system. The project emphasizes modular software design, subsystem validation, and first-principles engineering.

## Hardware Integration

The platform is built around a dual-power architecture in which the motor subsystem and control electronics operate from separate regulated paths while sharing a common ground reference.

The ESP32 performs state estimation and control computation using inertial measurements acquired from the MPU6050. Motor commands generated by the balance controller are transmitted to the L298N motor driver, which actuates the encoder-equipped N20 gear motors. Wheel encoder feedback provides velocity measurements for the inner control loop, enabling cascaded control of both body attitude and wheel motion.

The mechanical structure is implemented using a custom MDF chassis designed to maximize center-of-mass height while maintaining accessibility to the electronics and power subsystem.

## Software Architecture

```text
Sensor Layer
     ↓
State Estimation
     ↓
Balance Controller
     ↓
Velocity Controller
     ↓
Motor Driver
     ↓
Actuation Layer
```

The software is decomposed into independent modules responsible for sensing, estimation, control, and actuation.

### Sensor Module

- MPU6050 communication
- Accelerometer acquisition
- Gyroscope acquisition
- Sensor calibration
- Attitude estimation

### Motor Module

- Encoder processing
- Velocity estimation
- PWM generation
- Direction control
- Motor abstraction

### Balance Controller

The outer control loop computes corrective actions from the estimated body attitude and generates wheel commands required to maintain stability.

### Velocity Controller

The inner control loop regulates wheel velocity using encoder feedback.

```text
        ↓

        ↓
Velocity Controller
        ↓
Motor PWM
        ↓
Motor Driver
        ↓

```

The system employs a cascaded control architecture in which the outer loop stabilizes body attitude while the inner loop regulates wheel velocity.

---

## Sensor Fusion

Attitude estimation is performed using inertial measurements from the MPU6050.

The estimation process combines:

- Accelerometer measurements
- Gyroscope measurements
- Bias compensation
- Complementary filtering

This approach provides stable orientation estimates while reducing long-term drift and high-frequency sensor noise.
